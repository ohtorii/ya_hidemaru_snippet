==============================================================================
おまけスクリプト
==============================================================================

＊概要
    誰得なスクリプトです。
    snippets以下へ置くと邪魔になりそうなので、このディレクトリへ置いています。


＊スクリプトの紹介
    説明        ｜コマンドプロンプトの結果を得る。
    使用例      ｜@cmd dir *.txt /s
    ファイル名  ｜@cmd.%._.mac
    ------------------------------------------------------------------------
    説明        ｜引数の時を繰り返す。罫線を引くのが楽になる。
    使用例      ｜@rep 40 //
    ファイル名  ｜@rep.%._.mac


＊インストール方法
    snippets\text-mode\ 以下へコピーして下さい。
